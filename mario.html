Mario Math

Choose Your Character

Choose your favorite character and start saving the day! Mario math is an 2D GUI game designed to help children practice arithmetic while being engaged in a progressive story reliant on their correct answers.

Help Save Princess Peach!

Princess Peach has been captured and it's up to your wits and courage to save her from Lakitu. Unfortunately, a series of some angry, arithmetic goomba's are standing in your way. Answer each question correctly and in time and you can help save the day!

It's Math Madness!

Each Goomba has its on problem to solve. You can choose between Angry Addition, Marvelous Multiplaction, Deceptive Division, or Sneaky Subtraction as well set a number range.

How Mario Math Was Created

Mario Math was written in Java with the swing library used for GUI elements. It was invisioned to provide an interactive and engaging way for my baby cousin to learn his arithmetic. We shared a common interest in Super Mario Bros and the Nintendo world so this was a no-brainer. I had learned Java in my highSchool computer science classes and have demonstrated my learning through use of object oriented programming, memory management, and logical thinking.
